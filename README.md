# 🖐️ Virtual Keyboard using Computer Vision

🚀 I’m excited to share one of my key projects in Computer Vision! 🤖👁️✨

This project implements an **interactive virtual keyboard** that allows users to type using hand gestures only, without any physical contact.

---

## 🎯 Project Overview

The system uses **OpenCV** and **MediaPipe** to detect hand landmarks in real time.  
The index finger and middle finger 👆✌️ are used to simulate clicking on virtual keys displayed on the screen.

Each detected gesture is processed instantly, and the typed text appears in real time on the interface 📝⚡.

---

## 🛠️ Technologies Used

- Python 🐍
- OpenCV 🎥
- MediaPipe ✋
- CVZone 📦

---

## ✨ Features

- Real-time hand tracking using webcam
- Virtual keyboard overlay on screen
- Gesture-based key pressing (index + middle finger)
- Instant text display output
- Smooth and interactive user experience
